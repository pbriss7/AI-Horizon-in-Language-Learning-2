# **Atelier : design d'invite et création d'un assistant correcteur**

Ce projet contient les documents d'accompagnement pour un atelier de 90 minutes destiné à des professeur·es de français langue seconde (FLS/FLE).

Objectifs principaux :

-   Comprendre les principes de base du *prompt engineering* (design d'invite).

-   Savoir rédiger un message système clair pour transformer un LLM généraliste en assistant spécialisé.

-   Découvrir la notion d'« agent » et observer le comportement d'un agent correcteur (RAG) centré sur la correction linguistique.

## **Structure du dépôt**

-   `exercices/` : exemples de textes à corriger et de message système.
-   `Ollama_AnythingLLM_Atelier1/` : présentation du premier atelier portant sur Ollama et les LLM ouverts.

## **Pré-requis pour les participant·es**

-   Avoir accès soit :

    -   à CoPilot (Entreprise) à travers un navigateur, ou

    -   à un environnement avec un modèle ouvert (p. ex. Ollama + AnythingLLM).
